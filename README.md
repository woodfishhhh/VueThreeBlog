# VueCubeBlog

Static Vue 3 blog SPA with a Three.js homepage scene, generated Markdown content, and glassmorphism UI.

## Stack

Vue 3, Vite, TypeScript, Pinia, Vue Router, Three.js, Tailwind CSS 4, Vitest, Playwright.

## Quick Start

```bash
npm install
npm run dev
npm run build:with-content
```

See `docs/` for project notes and `docs/plan.md` for the migration plan.
